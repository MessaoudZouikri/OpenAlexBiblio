## Citation Formats

### BibTeX

```bibtex
@software{zouikri2026bibliometric,
  author = {Zouikri, Messaoud},
  title = {Bibliometric Pipeline: {P}opulism Studies},
  year = {2026},
  month = {04},
  version = {1.1.0},
  url = {https://github.com/MessaoudZouikri/econoPoPop},
  note = {A modular, reproducible pipeline for bibliometric analysis of populism using OpenAlex},
  contact = {econoPoPop@proton.me}
}
```

### APA Style

Zouikri, M. (2026). Bibliometric pipeline: Populism studies (Version 1.1.0) [Software]. Retrieved from https://github.com/MessaoudZouikri/econoPoPop

### MLA Style

Zouikri, Messaoud. "Bibliometric Pipeline: Populism Studies." GitHub, 2026, https://github.com/MessaoudZouikri/econoPoPop.

### Chicago Style

Zouikri, Messaoud. "Bibliometric Pipeline: Populism Studies." GitHub. Accessed [date], https://github.com/MessaoudZouikri/econoPoPop.

### IEEE Style

[1] M. Zouikri, "Bibliometric Pipeline: Populism Studies," GitHub, 2026. [Online]. Available: https://github.com/MessaoudZouikri/econoPoPop. [Accessed: date].

## Citation via GitHub

GitHub provides automatic citation format suggestions. You can:

1. **On GitHub**: Go to the repository home page
2. **Click**: "Cite this repository" button (in the About section)
3. **Copy**: Your preferred citation format

The repository is also available as a CFF (Citation File Format) file: `CITATION.cff`

## Citing Specific Components

If you're citing specific analysis or methods, you can reference the associated papers/reports:

### Code Quality Analysis & Enhanced Metrics

```bibtex
@techreport{zouikri2026analysis,
  author = {Zouikri, Messaoud},
  title = {Comprehensive Code Quality Analysis, Domain Taxonomy, and Enhanced Metrics},
  year = {2026},
  month = {04},
  type = {Technical Report},
  url = {https://github.com/MessaoudZouikri/econoPoPop/blob/master/CODE_QUALITY_ANALYSIS.md}
}
```

### Domain Taxonomy Framework

```bibtex
@misc{zouikri2026taxonomy,
  author = {Zouikri, Messaoud},
  title = {Domain Taxonomy Update Template for Research Collaboration},
  year = {2026},
  month = {04},
  url = {https://github.com/MessaoudZouikri/econoPoPop/blob/master/TAXONOMY_UPDATE_TEMPLATE.md},
  note = {Framework for enriching domain taxonomy with researcher feedback}
}
```

### Implementation & Metrics Guide

```bibtex
@misc{zouikri2026implementation,
  author = {Zouikri, Messaoud},
  title = {Implementation Summary and Enhanced Coupling Metrics},
  year = {2026},
  month = {04},
  url = {https://github.com/MessaoudZouikri/econoPoPop/blob/master/IMPLEMENTATION_SUMMARY.md},
  note = {4-phase roadmap with 4 interpretable cross-domain coupling metrics}
}
```

## In-Text Citation Examples

### First mention in text:
"We used the Bibliometric Pipeline for Populism Studies (Zouikri, 2026) to analyze bibliographic coupling and domain interactions in the populism literature."

### With version reference:
"Following Zouikri's (2026) pipeline (v1.1.0), we classified papers into 21 subcategories across 4 domains using a 3-stage hybrid approach."

### With specific component:
"We employed the Association Strength metric (Zouikri, 2026), which normalizes coupling strength against random expectation following VOSviewer methodology."
